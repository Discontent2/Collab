# Contributing to Meridian Prime Collaborator Intake

Thank you for contributing to Meridian Prime.

This repository is an intake space. Submissions are not canon and are not sandbox-approved until the owner explicitly promotes them.

## Contribution Status

Every contribution starts as:

```text
External Contribution / Non-Canon / Unreviewed
```

## Before You Submit

Please make sure your submission:

```text
is your own original work or properly credited public-domain / licensed material
labels AI assistance or AI generation
uses a template when possible
includes tags and intended use
avoids revealing protected mysteries
does not overwrite existing canon
is comfortable being revised, remixed, or rejected by the owner
```

## Required Metadata

Every submission should include:

```text
Title
Contributor
Submission Type
Date
Status
Canon Risk
Spoiler Risk
AI Use Disclosure
Rights / Originality Statement
Summary
Details
Tags
Promotion Recommendation
```

## AI Use Disclosure

Use one of these:

```text
No AI assistance
AI-assisted brainstorming
AI-assisted drafting
AI-generated image
AI-generated text requiring human review
Mixed human / AI contribution
```

## Review Outcomes

The owner may mark a submission as:

```text
accepted-for-sandbox
needs-revision
parked-for-later
rejected-or-parked
protected-mystery-risk
candidate-canon-later
```

## Contributor Rights Placeholder

By submitting here, you are indicating that you have the right to share the material for review and that you understand it may be edited, rejected, remixed, or used as inspiration if accepted by the owner.

This is a working creative pipeline notice, not legal advice. The project owner should establish a formal contributor rights policy before opening broad public collaboration.
