# Promotion Path

Meridian Prime uses a staged promotion model.

```text
Collaborator Intake → Sandbox → Canon
```

## Stage 1: Collaborator Intake

Repository:

```text
Discontent2/Collab
```

Status:

```text
External Contribution / Non-Canon / Unreviewed
```

Purpose:

```text
collect ideas
preserve contributor credit
review for usefulness
screen for protected mystery risk
screen for canon conflict
```

## Stage 2: Owner Review

Possible outcomes:

```text
Accepted for Sandbox
Needs Revision
Parked for Later
Rejected or Parked
Protected Mystery Risk
Candidate Canon Later
```

## Stage 3: Sandbox Promotion

Repository:

```text
Discontent2/Discontent2-meridian-prime-comic-sandbox
```

Status:

```text
Sandbox Material / Non-Canon / Requires Canon Review Before Promotion
```

Only the owner may promote intake material to sandbox.

## Stage 4: Canon Review

Repository:

```text
Discontent2/meridian-prime
```

Status options:

```text
Candidate Canon
Canon with Revision
Canon
Rejected from Canon
Hold in Sandbox
```

Nothing in this repo moves directly to canon.

## Promotion Packet Checklist

Before moving a submission to sandbox, prepare:

```text
source file path in Collab repo
contributor credit
submission summary
canon risk assessment
protected mystery assessment
recommended sandbox folder
required edits before promotion
owner approval note
```

## Promotion Packet Template

```markdown
# Promotion Packet: <Title>

**Source Repo:** Discontent2/Collab

**Required Edits:** <list>  
**Credit Notes:** <notes>  
```
