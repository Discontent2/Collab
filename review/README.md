# Review Board

This folder tracks owner review outcomes for collaborator submissions.

## Review Buckets

```text
review/accepted-for-sandbox/
review/needs-revision/
review/parked-for-later/
review/rejected-or-parked/
review/protected-mystery-risk/
review/candidate-canon-later/
```

Use small review notes or promotion packets in these folders rather than editing contributor submissions directly unless revision is requested.

## Standard Review Note

```markdown
# Review: <Submission Title>

**Source File:** <path>  
**Contributor:** <name or handle>  
**Review Outcome:** <outcome>  
**Reviewer:** Owner / Assistant-assisted owner review  
**Date:** <date>  

## Summary

## Strengths

## Concerns

## Canon Risk

## Protected Mystery Notes

## Sandbox Promotion Recommendation

## Required Edits

## Credit Notes
```
