# Meridian Prime Canon Guardrails for Collaborators

This file protects the deep machinery of Meridian Prime.

Contributors are welcome to create pressure, rumor, contradiction, false records, unreliable testimony, propaganda, local superstition, and side-story tension.

Contributors may not solve protected mysteries.

## Protected Mysteries

Do not define, reveal, overwrite, or solve:

```text
the Core
Dry Port truth
Dry Gate truth
gateway mechanics
Contact Frame mechanics
protected route topology
Ildi’s restricted truth
Conjugate Chimera truth
Rob Holliday’s death-chain
the NCI assassin
Book One answers
```

## Allowed Near Protected Mysteries

Allowed:

```text
rumors
false records
conflicting maps
classified pressure
misunderstood witnesses
corporate euphemisms
religious or folk interpretations
failed experiments
partial clues
local names for things not fully understood
```

## Forbidden Without Owner Approval

Forbidden:

```text
This is the true Dry Gate.
Here is how the gateway works.
The Core is actually...
Ildi was created by...
The NCI assassin is...
Rob died because of this full chain...
Book One should reveal...
```

## Safe Language

Use:

```text
suspected
rumored
classified
unverified
misrecorded
candidate
false label
pressure zone
witness claim
fragment
```

Avoid:

```text
confirmed
true
actual
final
explained
solved
canon answer
```

## Owner Review Rule

If a submission touches a protected mystery, mark it:

```text
Canon Risk: Protected Mystery Risk
Spoiler Visibility: Restricted
Promotion Recommendation: Owner review before sandbox
```
