# Start Here: Meridian Prime Collaborator Intake

This repo is the intake gate for Meridian Prime collaborators.

**Status of everything here:** External Contribution / Non-Canon / Requires Owner Review

## Fast Rule

```text
Ideas enter here.
Curated material moves to sandbox.
Only reviewed material can ever move to canon.
```

## Promotion Chain

```text
Discontent2/Collab
↓ owner review
Discontent2/Discontent2-meridian-prime-comic-sandbox
↓ canon review
Discontent2/meridian-prime
```

## Contributor Workflow

1. Pick a template from `templates/`.
2. Fill it out honestly.
3. Save it under the matching `submissions/` folder.
4. Label AI-generated or AI-assisted material.
5. Avoid protected mysteries unless you are only creating pressure or rumor.
6. Wait for owner review.

## Owner Review Outcomes

```text
accepted-for-sandbox
needs-revision
parked-for-later
rejected-or-parked
protected-mystery-risk
candidate-canon-later
```

## The Golden Guardrail

```text
Contributors may create pressure near protected mysteries, but may not solve them.
```

Protected mystery examples include the Core, Dry Port truth, Dry Gate truth, gateway mechanics, Contact Frame mechanics, Ildi’s restricted truth, Conjugate Chimera truth, Rob Holliday’s death-chain, the NCI assassin, and Book One answers.
