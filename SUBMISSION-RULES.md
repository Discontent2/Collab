# Submission Rules

## Prime Rule

```text
This repository is intake only.
```

No submission is canon. No submission is sandbox-approved until the owner explicitly promotes it.

## Allowed Submission Types

```text
story seed
NPC
location
species / creature
faction
art prompt
music prompt
game mechanic
map / cartography idea
worldbuilding fragment
comic concept
tooling / generator idea
```

## Required Fields

Each submission should include:

```text
Title
Contributor
Submission Type
Date
Status
Canon Risk
Spoiler Risk
AI Use Disclosure
Rights / Originality Statement
One-Paragraph Summary
Full Submission
Tags
Promotion Recommendation
```

## Canon Risk Levels

Use one:

```text
Low
Moderate
High
Protected Mystery Risk
Book One Risk
Canon Conflict Risk
Unknown / Needs Owner Review
```

## Spoiler Visibility

Use one:

```text
Open
Subtext Only
Restricted
Do Not Reveal
Protected Mystery Hint
```

## Do Not Submit

```text
material copied from copyrighted sources you do not control
unlabeled AI output
canon rewrites without permission
answers to protected mysteries
real-person private information
malware or executable code without clear purpose
harassment or hateful content
```

## File Naming

Use lowercase kebab-case:

```text
submissions/npcs/blue-signal-witness.md
submissions/locations/reed-shrine-underpass.md
submissions/maps/hydropolis-basin-draft.md
```

## Submission Status Header

Use this at the top of new submissions:

```markdown
**Status:** External Contribution / Non-Canon / Unreviewed  
**Owner Review Required:** Yes  
**Promotion Target:** Sandbox only, unless later approved for canon review  
```
